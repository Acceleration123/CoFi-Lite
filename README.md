- [**2026-7-7**] The manuscipt is accepted by IEEE SPL.

## ⚠️ Code Release Status

Thank you for your interest in our work! 

Due to commercial confidentiality and internal intellectual property (IP) review processes associated with our industry partners, the full source code and pre-trained models are currently **undergoing compliance checks and cannot be released immediately**. 

We are working hard to clear these hurdles. We plan to release the inference code and a pre-trained model for academic usage by **1st January, 2027**. 

We sincerely apologize for the delay. In the meantime, please **Star** or **Watch** this repository to receive notifications when the code becomes available.
